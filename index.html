URL Shortener!

Example creation usage:

https://freecodecamp-lubien-shurl.herokuapp.com/new/https://google.com
To accept invalid URLs, use:
https://freecodecamp-lubien-shurl.herokuapp.com/invalid/invalid-url

Creation output:

{ "original_url": "http://google.com/", "short_url": "https://freecodecamp-lubien-shurl.herokuapp.com/{integer number}" }
